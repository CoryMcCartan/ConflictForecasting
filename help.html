Working with Files

The "New" button creates a new, blank, file.

The "Open" button opens a previously created file.

The "Save" button saves the current file. If the file has not yet been saved, a prompt will be displayed asking for a file name.

The "Save As" button allows the current file to be saved again under a different name.

The "Manage Files" button shows a dialog where previously created files can be deleted.

The "Delete File" button shows deletes the current file from storage and clears the screen.

The "Import" button imports a file as text.

The "Share" button shares a file as text.

Managing Players

Pressing "Add Player" creates a new entry in the player table, which can be edited freely. All numeric values must be between 0 and 100.

Pressing "Remove Player" removes the player where the cursor is, or, if there is no cursor in a player entry, removes the last player in the table. Player deletion cannot be undone.

Pressing "Clear All" removes all players. This action cannot be undone.

Customizing Page Layout

Clicking the various buttons on this tab will hide or show their corresponding sections on the page. The notes section is a place for typing notes or analysis about the file.

Pressing "Print" will print the page. To save as a PDF, use the "Save as PDF" printer option in the print dialog.

Special Game Options

The "Shock Salience" button sets the probability that a given player will be given an entirely new salience value on a given round.

The "Force Length" button fixes the number of bargaining rounds before the simulation begins.

The "Default Value" button sets the position value that will be used if a player vetoes the agreement, or if no agreement can be reached.

The "Add Scale Point" button creates a new scale point, which is a text label for a given position value. When the final outcome is determined, the label with the closest position value is selected to represent the outcome. To have the software select the text labels when it runs the forecast, ensure that the "Scale" button is selected in the page tab. To hide the scale section for printing, simply click the "Scale" button again before printing but after running the forecast.
